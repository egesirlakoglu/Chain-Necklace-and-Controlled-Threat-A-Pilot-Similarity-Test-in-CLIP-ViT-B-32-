# Chain-Necklace-and-Controlled-Threat-A-Pilot-Similarity-Test-in-CLIP-ViT-B-32-
This project reports a small pilot similarity study using CLIP (ViT-B/32). Using 12 images and 8 text prompts, image-text and image-image cosine similarities were computed in the CLIP embedding space. The project shares stimuli links, similarity outputs, and figures. No human data or brain data were used.
